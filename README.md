📌 Loan Prediction MLOps Pipeline
🧠 Problem Statement

This project predicts whether a loan application will be approved based on applicant financial and personal data using a Machine Learning model deployed via Flask API.

⚙️ Workflow
Data preprocessing
Model training (ML algorithm)
Model serialization (model.pkl)
Flask API creation
Docker containerization
CI/CD automation using GitHub Actions
